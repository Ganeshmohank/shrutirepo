Publications


Spring '24

A hierarchical approach for the diagnosis of sleep disorders using convolutional recurrent neural network.
This study proposes a hierarchical approach using single-channel EEG data to automatically classify sleep disorders and CAP phases. Combining LSTM and CNN achieved high accuracy (91.45%-93.31%) in distinguishing healthy-unhealthy sleep patterns and identifying specific disorders, emphasizing the importance of phase B in detection.

Spring '23

Automated classification of cyclic alternating patterns sleep phases in healthy & sleep-disordered subjects using convolutional neural networks.
This study introduces a deep learning model using 1D CNNs for automated classification of CAP phases (A and B) from single-channel EEG recordings. Achieving up to 82.21% accuracy for disorders like narcolepsy and insomnia, the model assists in reducing manual errors and improving sleep monitoring for medical professionals.
